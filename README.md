# NPA Risk Predictor

An AI-powered web application for predicting Non-Performing Assets (NPAs) in loan portfolios.

## Features

- **Real-time Prediction**: Instant risk assessment for loan applications
- **Comprehensive Dashboard**: Analytics and insights into loan portfolio
- **API Support**: REST API for integration with other systems
- **Detailed Reports**: Comprehensive risk analysis with recommendations
- **Responsive Design**: Works on desktop and mobile devices
